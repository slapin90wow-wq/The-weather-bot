# currency-converter
Простой бот, позволяющий получить информацию о погоде в любом городе мира на утро день и вечер.
Дополниельно выдается рекомендация по одежде на разное время суток.
Данные о погоде подгружаются по API из OpenWeatherMap 

Бот отвечает только во время работы терминала.
